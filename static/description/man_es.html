Configuración de Mercado Pago

En un ambiente productivo

  1. Inicia sesión con tu cuenta de Mercado Pago en https://www.mercadopago.com.mx/developers
  2. Selecciona Crear nueva aplicación en el apartado Tus integraciones.
  3. Crea una aplicación con los datos que consideres apropiados, en el apartado Elige el producto que vas a integrar selecciona Checkout Pro.
  4. Una vez creada la aplicación podrás seleccionarla e ingresar a Credenciales de producción
  5. Copia el valor de Access Token al campo correspondiente de la configuración de Mercado Pago en Odoo. Este campo es requerido en Odoo cuando el Medio de Pago está en Habilitado

En un ambiente de pruebas

  1. Una vez creada la Aplicación en Mercado Pago (pasos 1 al 3 de la configuración en un ambiente productivo), elige la opción Credenciales de Prueba
  2. Finalmente copia el valor de Acces Token en el campo Test Acces Token en la configuración de Mercado Pago en Odoo. Este campo es requerido en Odoo cuando el Medio de Pago está en Modo Test
