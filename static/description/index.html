Mercado Pago Payment Integration

This module enables Mercado Pago as a payment method on the Odoo Web Site.

An easy configuration is required, you must enter the credentials provided by Mercado Pago.

Step 1:

Go to Web Site > Configuration > Payment Acquirers.

Step 2:

Look for Mercado Pago option and set as Enabled. Then set the following fields:

  • App ID
  • Access Token
  • Test Access Token
If you need help to get this information, click on "How to set up your Mercado Pago account"?.

Opcionaly:

Customize the messages that will be shown to your customers.

Step 3:

Now, when making a purchase on the Web Site, you will be able to select the payment method Mercado Pago.
